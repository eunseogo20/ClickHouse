# 비교 함수 {# comparison-functions}

비교 함수는 항상 0 또는 1 (Uint8)를 돌려줍니다.

다음의 유형을 비교할 수 있습니다 :

- 숫자
- 문자열과 고정 문자열
- 날짜
- 날짜와 시간

각 그룹이 아닌 다른 그룹 사이.

예를 들어, 날짜와 문자열을 비교할 수 없습니다. 문자열을 날짜로 변환하려면 함수를 사용해야합니다.

문자열은 바이트 단위로 비교됩니다. 짧은 문자열은 그래서 시작 적어도 한 개 이상의 문자를 포함하는 모든 문자열보다 작습니다.
