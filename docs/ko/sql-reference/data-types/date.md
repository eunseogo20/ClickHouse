# 날짜 {#date}

날짜 형식입니다. 1970-01-01의 일수가 2 바이트의 부호없는 정수로 저장됩니다. UNIX 시간의 시작부터 변환 단계에서 상수로 정의되는 상한 임계 값 사이의 값을 저장할 수 있습니다 (현재는 2106 년까지이지만 일년 분을 완벽하게 지원하는 것은 2105 년까지 입니다).

날짜 값은 시간대없이 저장됩니다.
